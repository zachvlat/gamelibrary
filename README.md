# Game Library

This is a simple game library application.

## Installation

To install the application, clone the repository and run `npm install`.

```
git clone https://github.com/zachvlat/gamelibrary.git
cd gamelibrary
npm install
```

## Usage

To run the application, run `npm start`.

```
npx expo start
```
